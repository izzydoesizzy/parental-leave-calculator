# Parental Leave Calculator

> Calculate your parental leave entitlements, benefits, and plan your time off with confidence.

[![GitHub Pages](https://img.shields.io/badge/demo-live-brightgreen)](https://izzydoesizzy.github.io/parental-leave-calculator/)

---

## What Is This?

A free, open-source calculator that helps expecting and new parents understand their parental leave options. Covers federal and provincial/state entitlements, employer top-ups, and helps you plan financially for time away from work.

### Planned Features

- **Leave entitlement calculator** — federal (EI) + provincial benefits by jurisdiction
- **Employer top-up estimator** — model different top-up scenarios
- **Financial planner** — budget projections during leave period
- **Timeline builder** — visualize your leave schedule (maternity, parental, shared)
- **Province/state selector** — jurisdiction-specific rules and maximums
- **Export** — save your plan as PDF

## Tech Stack

- HTML, CSS, JavaScript
- GitHub Pages
